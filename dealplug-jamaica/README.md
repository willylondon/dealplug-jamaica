# DealPlug Jamaica 🇯🇲

Jamaica's trusted deals platform - curating the best discounts from Temu & AliExpress.

## Features

- 🔥 Daily curated deals with 50-80% savings
- 💰 USD pricing for international shopping
- 📱 Mobile-responsive design
- 📰 Blog section with shopping tips
- 🔔 Email newsletter subscription
- 📲 Social media integration (Twitter/X, TikTok, Telegram)

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Deployed on Vercel

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Deployment

This project is configured for automatic deployment on Vercel.
